# plexRename
Renames shows for Plex compatibility

Requires shows to be in correct numeric order, or they will be incorrectly relabeled.

########## INTENDED FILE STRCUTURE USAGE ##########

# . (os.getcwd())
# |-- Season1
# |   |-- Episode1.mkv
# |   |-- Episode2.mkv
# |   |-- Episode3.mkv
# |   |-- Episode4.mkv
# |-- Season2
# |   |-- Episode1.mov
# |   |-- Episode2.mov
# |   |-- Episode3.mov
# |   |-- Episode4.mov
# |-- Season3
# |   |-- Episode1.mp4
# |   |-- Episode2.mp4
# |   |-- Episode3.mp4
# |   |-- Episode4.mp4
